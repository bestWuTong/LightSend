功能特点

轻传是一款基于 Flutter 开发的跨平台云端文件传输工具，通过用户自己的 OneDrive 个人账户或 WebDAV 服务在不同设备之间传递文件。OneDrive 使用应用专属目录存取文件，WebDAV 可连接自有服务器或兼容服务。无论是在桌面端还是移动端，轻传都能提供便捷的上传、下载和文本发送体验。

跨平台支持

支持 Windows、Android 和 Linux，在不同设备间自由传输

云端存储

支持个人 OneDrive 和 WebDAV，使用自己的云端账号或服务器存取文件

应用专属目录

OneDrive 使用应用专属目录，减少对网盘中其他文件的影响

多配置切换

可创建多个 OneDrive 或 WebDAV 配置，并在配置列表中快速切换

优雅界面

现代化的设计风格，支持自定义主题

快速分享

应用小而美，启动迅速；集成 Windows “发送到”和 Android 系统分享
